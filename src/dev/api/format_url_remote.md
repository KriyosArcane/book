# format_url_remote

Convert shellcode url to another format. (`Remote` Only)

## Function Name

format_url_remote

## Input

```json
{
  "url": ""
}
```

`url` is a string, which is the original shellcode url.

## Output

```json
{
  "formated_url": ""
}
```

`formated_url` is a string, which is the converted shellcode url.
